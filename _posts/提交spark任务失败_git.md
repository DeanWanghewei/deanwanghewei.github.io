## 分析

```
CDH的hive，spark。yarn组件都有gateway  模式。该机器上未装有yarn .spark hive 的这些组件。在提交spark任务的时候。由于没有装组件。所以就没有这些依赖包部署过来。
```

## 解决方案

